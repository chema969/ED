◆ crossProduct()

Vector3D ed::Vector3D::crossProduct ( ed::Vector3D const &  v) const

Calcula el producto vectorial entre dos vectores.

Parámetros
vVector con el cual se multiplica para conseguir el producto vectorial entre ambos
Devuelve
\( \vec{w}\)=Producto vectorial de \(\vec{u}\) y \(\vec{v} \)

\( \vec{w}=\vec{u}\)^ \(\vec{v}=w_1\vec{i}+w_2\vec{j}+w_3\vec{k}\) donde
\(w_1=u_2*v_3-u_3*v_2\)
\(w_2=-u_1*v_3+u_3*v_1\)
\(w_3=u_1*v_2-u_2*v_1\)
